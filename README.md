## Getting Started Development

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000)

## Building app

```bash
npm run build
#and after build finish
npm run start
```

Open [http://localhost:3000](http://localhost:3000)
